# Markdown Previewer

A live markdown previewer with a split-pane editor built with pure HTML, CSS, and JavaScript.

## Features

- **Split-pane layout** — editor on the left, live preview on the right
- **Custom markdown parser** — no external libraries
- **Supported syntax:** headings, bold, italic, strikethrough, code blocks, inline code, links, images, ordered/unordered lists, blockquotes, horizontal rules
- **Live preview** as you type
- **Copy HTML** button to copy rendered output
- **Clear** button to reset the editor
- **Dark theme** with syntax-like styling
- **Responsive** — stacks vertically on mobile

## Usage

Open `index.html` in any modern browser. Type markdown in the left pane and see the rendered preview on the right.

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox)
- Vanilla JavaScript (no frameworks or libraries)
